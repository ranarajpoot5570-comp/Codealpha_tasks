# CodeAlpha C++ Internship Tasks

This repository contains all the C++ projects completed for the **CodeAlpha internship**.
Each project is organized in its own folder with source code and a detailed README.

## Tasks Included

### 1. Banking System
Folder: `Banking System`  
- Console-based banking operations
- Features: account creation, deposit/withdraw, balance check, transaction history
- Uses OOP 

### 2. CGPA Calculator
Folder: `CGPA Calculator`  
- Calculates GPA and CGPA based on user input and credit hours
- Handles multiple subjects with different credit weights

### 3. Sudoku Solver
Folder: `Sudoku Solver`  
- Solves 9x9 Sudoku puzzles using the backtracking algorithm
- Validates input and displays solution in console

### 4. Login & Registration System
Folder: `Login Registration System`  
- Handles user sign-up and login validation
- Securely stores credentials using file handling

## How to Explore
1. Open each folder to access the source code and README  
2. Follow the instructions in each README to compile and run the programs
